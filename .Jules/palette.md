## 2026-07-19 - [System Tray Contextual Information]
**Learning:** For background system tray apps, requiring a click just to check the status increases friction. Surfacing contextual state via hover tooltips significantly improves peripheral awareness without demanding user interaction. Visual grouping (separators) within the menu also prevents misclicks on destructive/terminating actions.
**Action:** Always consider the idle/background state of tray applications and how to provide low-friction status updates (e.g., dynamic tooltips) and safe visual grouping.
